## Running Nginx having 4 Replicas

```
kubectl run nginx --image=nginx:latest --replicas=4
```

## Watch the pods

```
kubectl get pods -w
```

## Expose the ElasticSearch HTTP API port:

```

kubectl expose deploy/nginx --port 80

```
