About me

Bueno, esta página recoge mi actividad profesional relacionada con mi aprendizaje para el perfil de DevOps.
En todo caso tienes un resumen a continuación de lo que tienes aquí:

  • Kubernetes: sistema de código libre para la automatización del despliegue, ajuste de escala y manejo de aplicaciones en contenedores​ que fue originalmente diseñado por Google y donado a la Cloud Native Computing Foundation.
    Se crea un servidor K8S con despligue de correo electrónico, Jenkis, GitLab, AppMobile List y un Home Page Portfolio.
  • Jenkins: servidor de automatización de código abierto líder, proporciona cientos de complementos para respaldar la creación, implementación y automatización de cualquier proyecto.
    Se realiza el despliegue de los war con Groovy.
  • GitLab: es una suite completa que permite gestionar, administrar, crear y conectar los repositorios con diferentes aplicaciones y hacer todo tipo de integraciones con ellas, ofreciendo un ambiente y una plataforma en cual se puede realizar las varias etapas de su SDLC/ADLC y DevOps.
    • Se ha realizado las buenas practicas de Gitlab para gestionarlo.
    • Se ha realizado el repositorio de las aplicaciones de Kubernetes como gestor de versiones de cada una.
  • Jira: producto de software propietario para la gestión de proyectos, seguimiento de errores e incidencias.
  • Terraform: herramienta de aprovisionamiento de infraestructura donde puede almacenar la configuración de su infraestructura en la nube como códigos.
    El ciclo de vida de Terraform consiste en: init, plan, aplicary destruir.
  • Y la nube de AWS AWS (Amazon Web Services) es una plataforma de computación en la nube proporcionada por Amazon. Ofrece una amplia gama de servicios en la nube, como almacenamiento, bases de datos, redes y más. Con AWS, las empresas pueden construir y administrar aplicaciones de forma escalable y económica. También cuenta con una red global de centros de datos para alta disponibilidad y rendimiento. Es una opción popular para empresas y desarrolladores que desean aprovechar la potencia de la nube para sus proyectos. Con servicios como:
    • Amazon EC2 (Elastic Compute Cloud)
    • Amazon S3 (Simple Storage Service) and S3 Glacier
    • Amazon RDS (Relational Database Service)
    • Amazon VPC (Virtual Private Cloud)
    • AWS Lambda
    • Route 53
    • EKS
    • Block Level Storage
    • Cloud Front - Global content delivery network.
    • Track IAM & API Usage
